# 1. 项目启动
+ 安装依赖 `pip install -r requirements.txt`
+ 创建 `config_env.py`，内容和 `config_env_default.py` 的内容一样就可以。或者是修改 `settings.py` 文件里引用的 `config_env.py` 为 `config_env_default.py` 即可
+ 启动项目 `python manage.py runserver`
